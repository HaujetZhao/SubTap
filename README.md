# 字幕音频播放器 · 英语主动点读工具

一个纯前端网页工具，用**主动点读**的方式学英语：载入 SRT 字幕和音视频，点击任意句子播放对应片段，句子里的单词按难度分级**着色高亮**，一眼看出哪些词该关注。

## 为什么做这个

被动听英语听力容易犯困——听到不懂的句子才发现没注意字幕。这个工具反过来：**先主动读句子、再点听**，你想听哪句点哪句，词库还会把生词按难度标色，阅读时自然聚焦。

## 功能

- **三栏布局**：左栏（分级勾选 / 字幕微调 / 文件入口）、中栏（视频 + 句子列表）、右栏（当前句单词）
- **词库内置**：7 个分级（初中/高中/四级/六级/考研/托福/SAT）约 34000 词，无需上传
- **颜色高亮**：句子里的词按所属分级着色（绿→粉由易到难），**超纲**（不在词库的词）标灰
- **高亮跟随勾选**：取消某级，中栏不高亮、右栏不显示，一致联动
- **音频 / 视频统一**：音频时视频区收起不可展开；视频时默认展开为视窗一半、可拖拽调高度
- **字幕时间微调**：起始偏移、末尾延长、句末连接（解决长词被字幕提前切断）
- **精准播放**：点到句首、播到句尾自动停（`requestAnimationFrame` 精度，不咬下一句开头）
- **单文件交付**：构建产物是一个 `dist/index.html`（词库内置），可直接托管 GitHub Pages

## 快速开始

```bash
npm install
npm run dev      # 开 http://localhost:5173
```

打开后：选字幕 `.srt` → 选音频或视频 → 点句子播放。

## 构建

```bash
npm run build    # 产出 dist/index.html（单文件，含词库）
```

`dist/index.html` 可直接双击打开，或部署到任意静态托管（GitHub Pages 等）。部署到 `用户名.github.io/仓库名/` 子路径时，在 `vite.config.js` 设 `base: '/仓库名/'`。

## 使用说明

1. **载入文件**：左下"字幕 .srt"选字幕、"音/视频"选 mp3 或 mp4。
2. **点句子**：中栏点任意句，播放该句对应音视频片段，到尾自动停。
3. **分级勾选**：左栏勾选关注的分级（默认初中/高中/四级不勾选，其余勾选）。勾选的级别：中栏该级词着色 + 右栏显示释义。
4. **背景色总开关**："用背景色突出单词"可一键开关中栏着色（右栏不受影响）。
5. **字幕微调**（可选）：
   - 起始偏移：全局每句开始时间 ±秒
   - 末尾延长：每句多播 N 秒
   - 句末连接：每句播到下一句开头（解决长词被提前切断）

## 词库格式

`src/vocabulary.json` 是两级结构：

```json
{
  "初中": { "boat": "n. 船，小船", ... },
  "四级": { "compress": "v. 压缩", ... },
  ...
}
```

更新词库后跑 `npm run build` 重新内置即可。词库 keys 应全小写（查词时句子会转小写匹配）。

## 配色

集中在 `src/level-colors.js`，想调整某级颜色改这里：

| 初中 | 高中 | 四级 | 六级 | 考研 | 托福 | SAT | 超纲 |
|------|------|------|------|------|------|-----|------|
| 🟢 | 🔵 | 🔵 | 🟣 | 🟠 | 🔴 | 🩷 | ⚪ |

## 项目结构

```
├── index.html              # Vite 入口
├── vite.config.js          # Vue 插件 + singlefile 内联
├── package.json
├── src/
│   ├── main.js             # Vue 挂载
│   ├── App.vue             # 三栏布局 + 全局状态
│   ├── components/         # SettingsPanel / SentenceList / WordPanel
│   ├── styles.css
│   ├── vocabulary.json     # 词库（构建期内置）
│   ├── srt-parser.js       # 纯逻辑：SRT 解析
│   ├── word-lookup.js      # 纯逻辑：分词 / 分级 / 超纲
│   ├── vocab-store.js      # 纯逻辑：词库 + 勾选 + 查询
│   ├── player.js           # 纯逻辑：区间播放（rAF 精准停）
│   ├── subtitle-tweak.js   # 纯逻辑：时间微调
│   └── level-colors.js     # 分级配色
├── test.html               # 纯函数测试页（http 访问）
├── dist/index.html         # 构建产物（单文件）
└── docs/superpowers/       # 设计文档与实施计划
```

## 技术栈

Vue 3（`<script setup>` Composition API）+ Vite + vite-plugin-singlefile。纯逻辑层是框架无关的 ES module，可独立测试。

设计与实施计划在 `docs/superpowers/`（三个迭代批次：三栏布局、视频与微调、颜色高亮）。开发指南见 `CLAUDE.md`。
